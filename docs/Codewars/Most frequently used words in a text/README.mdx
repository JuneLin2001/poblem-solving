### 解題思路

使用 Counter 來計算單詞出現次數，再用正則表達式篩選出符合題意的格式。  
重點為：

1.  轉小寫
2.  忽略標點符號（但 won't 的標點不能被忽略掉）

最後轉換物件為陣列並依次數由大到小排序，並根據單詞總數決定要返回多少個單詞

```js
Object.entries(counter) // 將 Counter 轉成陣列
  .sort(([, a], [, b]) => b - a)
  .slice(0, Math.min(3, Object.keys(counter).length)) // 根據單詞總數動態調整要返回多少個單詞
  .map(([key]) => key);
```
