### 解題思路

```js
function lookSay(number) {
  const numStr = number.toString();
  let result = "";
  let count = 1;

  for (let i = 0; i < numStr.length; i++) {
    // 如果下一個數字跟現在的一樣，就累加
    if (numStr[i] === numStr[i + 1]) {
      count++;
    } else {
      // 否則就把目前的統計加進結果字串
      result += count.toString() + numStr[i];
      count = 1; // 重置計數
    }
  }

  return parseInt(result, 10); // 轉成數字回傳
}
```

### 心得

不使用 Object 的 Counter，而是簡化為一個數字。
