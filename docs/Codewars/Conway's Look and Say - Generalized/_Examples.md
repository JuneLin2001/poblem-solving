For example:

- Start with 1.
- There is one 1 --> 11
- Start with 11. There are two 1 digits --> 21
- Start with 21. There is one 2 and one 1 --> 1211
- Start with 1211. There is one 1, one 2, and two 1s --> 111221

Sample inputs and outputs::

- 0 --> 10
- 2014 --> 12101114
- 9000 --> 1930
- 22322 --> 221322
- 222222222222 --> 122
